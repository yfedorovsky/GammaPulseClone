This replay: 34 BULL signals across 15 days of 3 prior weeks — hit@any=68%, hit@50bp=56%.

### Limitations

- EOD-to-EOD returns, not intraday — live engine fires in 9:30-4:00 PM window
- Structural direction only — skips RTS, IVP, breadth, trend-day, Mir
- 10 tickers (mega-cap tech heavy) — no broader universe coverage
- 3 weeks — multiple regime cycles would take months of data

### Next-step upgrades (in priority order)

1. **Port 5-factor SOE scorer to replay** — adds S/R confluence,
   IV environment, macro. Would likely raise BULL hit rate by 5-10pp.
2. **Expand ticker set to 30-50** — include mid-cap themes (AXTI,
   AAOI, RDDT) where live-week edge was strongest
3. **Intraday sampling** — pull chain snapshots at 10:00, 14:00, 15:30
   using ThetaData /v3/option/snapshot/* endpoints for historical
   chain-at-time queries
4. **Backfill signal_outcomes** — run `compute_signal_outcomes()`
   on replay signals to produce full forward-return tables that
   the existing attribution pipeline consumes
