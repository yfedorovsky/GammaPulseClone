## 1. Exact Screening Criteria Pros Actually Use (The Confluence That Has Edge)

Run **bottom-up** (strong stocks first) then overlay sector/market.

### Core Filters (Trend Template + SEPA – mandatory)
- **Relative Strength (RS)**: ≥80 (ideally top 2–5% of market)  
  → IBD RS Rating or Mansfield RS line rising and outperforming SPY
- **Price vs Moving Averages (Trend Template)**:
  - Price > 50 SMA
  - Price > 150 SMA
  - Price > 200 SMA
  - 150 SMA > 200 SMA
  - 50 SMA > 150 SMA and 200 SMA
  - 200 SMA rising for ≥1 month (ideally 4+ months)
  - Price within 25% of 52-week high
- **Volume**:
  - 50-day avg volume ≥ 500k–1M shares
  - Increasing volume on up days / breakouts
  - Decreasing volume on pullbacks (accumulation)
- **Fundamentals (CANSLIM acceleration)**:
  - Quarterly EPS growth ≥25% YoY (accelerating)
  - Quarterly sales growth ≥25% YoY
  - Profit margins expanding
- **Sector/Industry Group**: Top-quartile RS
- **ADR%**: ≥2.0% (for option premium potential)

**Proven combination**: All Trend Template + RS ≥80 + liquidity + sector strength.  
This is what produced documented 220% annualized returns (Minervini).

## 3. Institutional Sector Rotation Process (Desk-Level)

1. Rank **9–11 SPDR Sector ETFs** weekly by 1-week / 1-month / 3-month RS vs SPY.
2. Focus only on **top 2–3 sectors**.
3. Drill into **individual leaders** inside those sectors using the stock filters above.
4. Rebalance weekly (or intra-week on strong moves).

This is exactly how hedge funds, prop desks, and multi-strat PMs build their weekly focus list.

## 4. Minimum Viable Screen (3–4 Filters → 10–20 Names/Week)

From ~3,000–4,000 stocks → 10–20 actionable names:

1. **RS ≥80** (or top 10–20% 1-mo/3-mo performance) → **#1 most important**
2. **Trend Template basics**: Price > 50 SMA **AND** > 200 SMA **AND** 200 SMA rising ≥1 month
3. **Liquidity**: 50-day avg vol ≥500k–1M **AND** ADR% ≥2.0%
4. **Sector filter**: Stock in top 3 SPDR sectors by RS

**Optional 5th**: Price within 25% of 52-week high.

Run nightly in TradingView / Finviz / TC2000 / Thinkorswim.

## 5. Options-Specific Filters (7-14 DTE Calls/Puts)

**Stock criteria stay 95% the same.** Add these for options overlay on $20K account:

- **Underlying liquidity**: Avg daily shares ≥1M
- **Option chain**:
  - Open interest ≥200–500 contracts per strike (ideally 1,000+)
  - Bid/ask spread ≤ $0.10–$0.20 **or** <5–10% of premium (ATM/near-ATM)
- **IV Rank**: <50–60% when **buying** (cheap extrinsic value)
- **ADR%**: Still ≥2.0% (must overcome theta in 1–5 days)

**No change** to RS / Trend Template / Volume – these still drive directional edge.

## 6. Daily 15-Minute Morning Workflow (Pre-Market Routine)

**Night before (after close)**: Run full scan → save watchlist.

**Morning routine (15 minutes max)**:

- **0–5 min**: Market + Sector Check  
  → SPY/QQQ futures + SPDR sector heatmap/RS ranking → lock top 2–3 sectors
- **5–10 min**: Review/Refresh Scan  
  → Flag pre-market movers/gappers that still meet criteria + volume spike
- **10–15 min**: Quick Chart Review + Entry List  
  → Pull daily/weekly charts on 10–20 names  
  → Mark VCP/breakout levels, 7–8% stops, targets  
  → Build final 3–5 actionable names with exact entry prices + alerts

**Result**: Execution-only in the morning. No emotional scanning.
